Below is a library of overlay designs and custom firmware for TouchPad.  To use an overlay, click on the design to download the .svg + .ino design files.  Next, print the design onto transparency film (around 100 micron is ideal) and cut/hole punch.  If the design includes a .ino file you can upload the custom firmware using Arduino, otherwise program TouchPad's keys with the corresponding functions.
