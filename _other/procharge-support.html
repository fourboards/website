User Guide


ProCharge has three modes, as shown in the table below. Use the mode button to switch between them.

When switching from standby to delay mode, the number of amber and red flashes corresponds to the current charge and delay time settings respectively.


Setting the delay and charge times - Start in standby mode to set the delay time or in delay mode to set the charge time.  Hold the mode button until the status LED has flashed amber (if setting the delay time) or red (if setting the charge time) the number of times corresponding to the number of hours required.

Default settings - When ProCharge is unplugged, the delay and charge times will revert to their default settings of a 6 hour delay and a 2 hour charge.
